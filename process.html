Process

How this website was built

Design choices

The visual representation of this website comes directly from a mood board I crafted. Below are the colors and feelings i used to craft this website. I used one main serif display typeface for headings and a clean sans-serif for body text to keep things modern,warm and luxurious.

My information is split across five pages so there is enough space to see how I used HTML and CSS to create this layout. Each page shares a consistent header, quick navigation, and closing footer.

Creation & layout decisions

  • Used flexbox for the header and basic layout.
  • Used CSS for project cards shown below.
  • Added media queries to switch from wide view to condensed views on smaller screens.
  • Made sure all images were set to a comfortable width level, this way they can adjust accordingly.

Build Timeline

  1. Step 1 - Mood & structure. Planned out the color palette, typography, and number of pages I would use.
  2. Step 2 - HTML skeleton. Used HTML to present information on each page in an organized fashion.
  3. Step 3 - Shared CSS. Created one external stylesheet with typography, flexbox, grid, and media queries to keep consistency.
  4. Step 4 - Interaction. Added a small JavaScript filter to all pages.
